\# IoTree MQTT Auto Discovery v2



Home-Assistant Add-on zur dynamischen Erzeugung von Geräten und Entitäten

aus IoTree MQTT Payloads – inkl. Konfig-UI im Sidebar-Panel.



\## Features

\- MQTT Auto Discovery

\- Alle message-Felder als Entitäten

\- Tenant-sichere Device-IDs

\- Aktivieren/Deaktivieren je Entity

\- Persistente Konfiguration

\- Read-only



\## Nutzung

\- Add-on installieren

\- MQTT läuft automatisch

\- Sidebar "IoTree MQTT" öffnen

\- Entitäten konfigurieren
